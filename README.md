# Julia-tutorials-for-ChEs
This repository contains tutorials for the Julia programming language that are especially tailored for chemical and biochemical engineers (ChEs and BioChEs). In addition, the tutorials shown here emphasize the use of ModelingToolkit.jl and the combination of symbolic and numeric programming/computation for solving problems of interest to ChEs and BioChEs. Combining symbolic and numeric programming enables the development of both more effective solution approaches and more efficient numerical code. In addition, a pure Julia, single-language approach facilitates coordinating the user, symbolic, numeric, and compiled layers of the software. Although many tutorials exist for the Julia programming language, none have the focus of this tutorial, and many are a little short on using effective pedagogical methods to make learning easy (or at least easier!) for the neophyte. The tutorials here seek to address these needs.
