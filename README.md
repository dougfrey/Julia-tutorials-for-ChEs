# Julia-tutorials-for-ChEs
This repository contains tutorials for the Julia programming language that are especially tailored for chemical and biochemical engineers (ChEs and BioChEs). In addition, the tutorials shown here emphasize the use of [ModelingToolkit.jl](https://docs.sciml.ai/ModelingToolkit/stable/) and the combination of symbolic and numeric appoaches for solving problems of interest to ChEs and BioChEs. Combining symbolic and numeric approaches enables both more effective solution methods and more efficient compiled numerical code for existing methods. In addition, a pure Julia, single-language approach helps to coordinate the various parts of the overall computational method. Furthermore, the use of ModelingToolkit.jl yields an acausal approach to simulations, similar to that used in [Modelica](https://modelica.org/). Acausual approaches to simulations are generally considered to be superior to causual approaches, such as used in [Simulink](https://www.mathworks.com/products/simulink.html) in the Matlab ecosystem, since they represent the underlying physics more efficiently.

There are several other packages in the Julia ecosystem that are built on top of ModelingToolkit.jl to take advantage of its features. Examples include [Catalyst.jl](https://docs.sciml.ai/Catalyst/stable/) (for simulating chemical reaction networks),  [MethodOfLines.jl](https://docs.sciml.ai/MethodOfLines/stable/) (for solving PDEs), [Conductor.jl](https://wsphillips.github.io/Conductor.jl/stable/) (for neuroscience applications), and [ProcessSimulator.jl](https://github.com/SciML/ProcessSimulator.jl) (for simulating chemical processes and which in an early stage of development). All of packages described above are free and open source; however, ModelingToolkit.jl is also the foundation for several state-of-the-art commercial software systems such as [Dyad](https://juliahub.com/products/dyad) (for AI-assisted industrial simulations). Familiarity with Modelingtoolkit.jl greatly facilitates using all of these packages, and several of them will be incorporated into this tutorial. 

Although many tutorials exist for the Julia programming language, none have the focus of this tutorial, and many are somewhat short on using effective pedagogical methods to make learning easy (or at least easier!) for the neophyte. The tutorials here seek to address these needs.
